## 2. Getting Started

This is the installation guide.

### 2.1. Install with pip

```shell
$ cd compas
$ pip install -e .
```

### 2.2. Run an Experiment

```shell
$ python compas/experiments/run_LSTM.py --config="compas/experiments/config/train_example.yaml"
```
